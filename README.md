# Simple autoscroller

Simple program that scrolls wheel when mouse XButtons holded
*why nobody done that before?*

## Usage

Run `autoscroll.exe`
Hold X1 button to scroll up
Hold X2 button to scroll down

## Compiling

- Download [C3](https://c3-lang.org)
- Run
```bash
$ c3c compile main.c3
```
- Done
